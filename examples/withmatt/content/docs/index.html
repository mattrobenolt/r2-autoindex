r2-serve

A static file server for Cloudflare R2.

GitHub · npm · live example bucket

What this example shows

  • Generated directory listings cached in R2.
  • Directory index files: this page is served from docs/index.html.
  • Fallback rewrites: /latest and /downloads/example.txt.
  • Basic Auth: /private/ uses demo:r2-serve.
  • Hidden paths: /hidden/ exists in R2 but is not listed.
  • Byte-range media serving: Big Buck Bunny MP4.
  • Custom 404/500 pages.

Minimal Worker

import { createR2ServeWorker } from "r2-serve";

export interface Env {
  BUCKET: R2Bucket;
}

export default createR2ServeWorker<Env>({
  bucket: (env) => env.BUCKET,
  indexes: ["index.html", "index.htm"],
});

Cloudflare resources

The Terraform module can create the R2 bucket, Queue, R2 event notification rules, Worker, R2 binding, Queue consumer, and custom domain in one block.

It uses the committed Worker bundle, so Terraform can deploy the standalone Worker without running npm or Wrangler.
